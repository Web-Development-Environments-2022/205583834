Web 1.0:

  • Mainly a content delivery network that show information to the users
  • The pages are static which means the users can only read from the website

Web 2.0:

  • Highlights user generated content (the users can take part and not just the website owner
  • The pages are not read only base and the user can interact with the website
  • Infrastrucure to known technologies such as P2P,social networks, etc.

Web 3.0:

  • Altering the web into database
  • The development includes also backend side and not just frontend like it used to be
  • include new features such as AI, stronger connectivity, 3D graphics, etc.
  • Give the opportunity to software agents to process the data and not just for human users

Use Case Example:

in search on the internet of some medicine:
   Web 1.0: The websites will be give information about the specific medicine (static pages). The user can't respond to the content directly from the internet
   Web 2.0: The website will show information about the medicine (like in web 1.0) but it will also contains comments of other users and the option for the user to comment by himself
   Web 3.0: The websites will display the information and will recommend us on other alternatives (based on recommendation systems and other AI methods).
